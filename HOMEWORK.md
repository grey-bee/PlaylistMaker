# Домашнее задание: Переход на Fragment Navigation

## Задача на спринт

Перевести приложение на новую архитектуру навигации с использованием фрагментов и Jetpack Navigation Component.

### Три основные части работы:

1. **Перенос логики из Activity во фрагменты**
2. **Подключение Jetpack Navigation Component**
3. **Реализация Bottom Navigation**

## Рекомендуемая последовательность действий

### Шаг 1: Перенос Activity → Fragment
- Начать с переноса логики из активити во фрагменты (см. тему 1 спринта - пример с поиском фильмов)
- **Важно:** Экран «Аудиоплеер» тоже нужно перевести на Fragment

### Шаг 2: Подключение Navigation Component
- Вернуться к теме 2 спринта и выполнить последовательность шагов для внедрения фреймворка

### Шаг 3: Реализация BottomNavigationView
- Настроить и подключить BottomNavigationView (см. текущую тему)

## Важные моменты

⚠️ **Что изменится:**

1. **Экран входа удаляется полностью** - больше не нужен
2. **Кнопка "Назад" убирается** на экранах:
   - Поиск
   - Медиатека
   - Настройки
3. **Клавиатура должна скрывать панель навигации** - настроить `windowSoftInputMode` в манифесте:
   ```xml
   android:windowSoftInputMode="adjustPan"
   ```
